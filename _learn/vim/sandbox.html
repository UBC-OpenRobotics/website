Practice vim commands freely. Use this sandbox to experiment with any vim commands you'd like to try.

Quick Reference:

vim filename - Open vim
:w - Save file (in vim)
:q - Quit vim
:wq - Save and quit
i - Insert mode
Esc - Normal mode
h j k l - Move cursor
x - Delete character
dd - Delete line
yy - Copy line
p - Paste
u - Undo
w - Next word
b - Previous word
0 - Start of line
$ - End of line
